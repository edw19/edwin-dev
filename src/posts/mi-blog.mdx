<Paragraph>
  He decidido crear mi blog personal para compartir mi experiencias como
  desarrollador, tenía como objetivo no usar tecnologías como wordpress no
  porque sean malas, más bien por aprender un tema nuevo.
</Paragraph>

<Title title="Objetivos para la creación de mi blog" />
<Paragraph>1. Usar NextJS como tecnología principal</Paragraph>
<Paragraph>2. Una facil gestión para próximas publicaciones"</Paragraph>

<Title title="ReactJS" />
<Paragraph>
  Esta biblioteca de javascript nos permite crear interfaces de usuario basado
  en un concepto que se lo conoce como compocisión, para mi esta es la mejor
  caracteristica de la biblioteca debido a que me ayudo a planificar la creación
  de componentes en base a un objetivo sacandole el mayor provecho a la
  tecnología.
</Paragraph>
<Title title="Nextjs" />
<Paragraph>
  Como bien lo indica su documentación oficial, NextJS es el framework de React
  que potencia todos los beneficios que ofrece esta biblioteca dandonos
  funcionalidas geniales como la generación estática o la generación de html del
  lado del servidor. NextJS es genial y es mi tecnología favorita en 2021.
</Paragraph>

<Title title="TailwindCss" />
<Paragraph>
  TailwindCss es un framework de css que esta enfocado principalmente en
  brindarnos clases pre-escritas para agilizar el tiempo de desarrollo, en lo
  personal me ha gustado mucho tailwindcss, permite una amplia customización y
  me permite agilizar el proceso de maquetación de un sitio web.
</Paragraph>

<Title title="MDX" />
<Paragraph>
  Buscando en el ecosistema sobre tecnologías que permitán escribir texto plano
  y reutilizar ese mismo archivo para diferentes propósitos me encontre con
  Markdown que es un lenguaje de marcado que tiene como objetivo conseguir la
  máxima legibilidad usando una sintáxis más sencilla.
</Paragraph>
<Paragraph>
  MDX es un formato que combina markdown y la sistáxis de JSX, esto nos permite
  utilizar componentes de react y renderizarlos facilmente.
</Paragraph>
